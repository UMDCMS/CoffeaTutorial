Introduction to Awkward arrays
================================

This is an gentler introduction to the peculiar syntax that is awkward arrays,
which sits at the heart of how coffea can be both efficient to execute for a
machines and efficient to write for a experienced programmer. For people coming
from a more traditional loop-based programming syntax, the syntax will take some getting use to, but this tutorial can hopefully help you understand how to understand the syntax and how to understand the various method.
